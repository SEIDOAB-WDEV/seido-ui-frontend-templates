WebApi consumer example

This example is part of the ASP.NET Web Development course, and shows how to combine ASP.NET MVC, Razorpages, which is C# and .NET backend, with css and javascript from Bootstrap in the frontend.

We build a Webapplication that communicates and consume the services of the with the goodfriends WebApi that we build in the .NET Data Access course.

Using the WebApi services, we read and manipulate the data in the code-first database we created and populated in the .NET Data Access course. Enjoy!
